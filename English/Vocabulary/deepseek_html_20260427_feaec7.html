🧩 6. Word Sum Strategy (break down like maths)

antediluvian → ante- (before) + diluvian (flood) = extremely old ✅

circumnavigate → circum- (around) + navigate (sail) = sail around ✅

benevolence → bene- (good) + vol (wish) + -ence (noun) = kindness ✅

inspect → in- (into) + spec (look) = examine carefully ✅

✅ Root quick reference: spec=see, dict=speak, port=carry, graph=write, aud=hear, bene=good, mal=bad
✅ Suffix word class: -tion/-ness = noun, -ous/-ful = adj, -ize/-en = verb, -ly = adverb
✅ Spelling rules: double consonant (run→running), drop E (hope→hoping), Y→I (happy→happiness)
✅ Watch prefix traps: invaluable≠not valuable, inflammable=flammable
✅ Word Sum: prefix + root + suffix = meaning. Eliminate wrong answers by part of speech.
