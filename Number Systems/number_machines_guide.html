What is a Number Machine? A number machine (or function machine) takes an input number, applies one or more mathematical rules, and produces an output number. Think of it like a food processor: bread goes in, toasting happens inside, toast comes out. Your job is either to find the output, find the input, or find the hidden rule.

One-Step Machines — Forward & Inverse

Forward working means you know the input and the rule — find the output.
Inverse (backward) working means you know the output and the rule — find the input. To go backwards, use the opposite operation:

  • × ↔ ÷

Missing Rule Problems

You are given the input and output but one or both rules are hidden. Your job is to find the missing operation.
🎯 Strategy for finding a missing rule
  1. Write down what you know: input, output, and any visible rules.
  2. If one step is missing, work out what the intermediate value must be (apply the known rule to the input, or reverse a known rule from the output).
  3. Now ask: what single operation turns the intermediate value into the target? Check all four: +, −, ×, ÷.
  4. Verify by running the input all the way through.

Input: 12 · Output: 4 · What is the rule?

Finding Two-Step Rules

Given multiple input/output pairs for the same machine, deduce both rules. This is harder because there are many possible pairs of operations — you must use the pattern in the data to narrow it down.
🎯 Rule-finding strategy
  1. Look at how the outputs change as inputs increase. If outputs change by a fixed multiple of the input changes, the first operation is likely multiplication.
  2. Estimate the multiplier: change in output ÷ change in input.
  3. Apply the multiplier to a known input. The difference between the result and the actual output reveals the additive/subtractive second step.
  4. Verify your full two-step rule against every pair provided.
  5. Alternative: If you suspect a rule like ×n, write: output = n × input + c, then use two pairs to solve for n and c.
Ex 1 Deduce two-step rule from pairs
The machine gives: 2→11, 5→23, 8→35. Find the two-step rule.
1
Outputs increase: 11 → 23 → 35 (up by 12 each time). Inputs increase: 2 → 5 → 8 (up by 3). 12 ÷ 3 = 4. Multiplier = ×4.
2
If first step is ×4: input 2 → 2 × 4 = 8. But output is 11. 11 − 8 = 3. Second step = +3.
3
Verify: 5 × 4 + 3 = 23 ✓ and 8 × 4 + 3 = 35 ✓.
Rule = ×4 then +3
Ex 2 Rule-finding with a subtraction second step
The machine gives: 3→4, 7→16, 10→25. Find the two-step rule.
1
Outputs: 4, 16, 25. Differences: 12, 9. Inputs: 3, 7, 10. Differences: 4, 3. Ratios: 12/4 = 3 and 9/3 = 3. Multiplier = ×3.
2
Apply ×3 to input 3: 3 × 3 = 9. Output is 4. 4 − 9 = −5. Second step = −5.
3
Verify: 7 × 3 − 5 = 16 ✓ and 10 × 3 − 5 = 25 ✓.
Rule = ×3 then −5
Ex 3 Rule-finding with decimals
The machine gives: 2→7, 4→13, 6→19. Find the rule.
1
Outputs increase by 6; inputs increase by 2. 6 ÷ 2 = 3. Multiplier = ×3.
2
Input 2 ×3 = 6. Output is 7. 7 − 6 = 1. Second step = +1.
3
Verify: 4 × 3 + 1 = 13 ✓. Now find the output for input 3.5: 3.5 × 3 + 1 = 10.5 + 1 = 11.5.
Rule = ×3 then +1

Targeted Exam Tips

🔁
Always check your answer by going forward

When you find an input by working backwards, put it back through the machine in the forward direction and confirm you get the original output. This takes 20 seconds and catches nearly all mistakes.

📋
Stuck on a missing rule? Work to the middle

If a rule is missing, figure out what the intermediate value must be. Either apply the known first step forward, or reverse the known last step backward. Then the question becomes: "What single operation takes me from A to B?" — try ×, ÷, +, − in turn.

🔢
When to use algebra vs arithmetic

For single-input machines, arithmetic (number-crunching step by step) is fastest. Use algebra (y = 5x + 2) only when you need to find a rule from multiple pairs or when the question gives you a formula. Writing y = n × input + c and solving simultaneous equations is a reliable fallback for tricky rule-finding questions.

⚠️
Reverse order AND reverse operation — both, always

The most common error in inverse working is reversing the operations but keeping the same order. For a machine with rules A then B, the inverse is opposite-B first, then opposite-A. Write it as a reversed list before you start calculating.

📐
Don't panic about decimals or negatives

If your working gives a decimal (like 7.5) at an intermediate step, carry it through exactly. Decimals are valid answers. Similarly, negative numbers follow the same rules — the inverse of "add −3" is "subtract −3", which equals "add 3". If in doubt, write it as an ordinary calculation on the page.

📊
For table questions: always check more than one row

A rule that works for row 1 might not work for row 2. You need the same rule to fit every row. If your first guess fails on row 2, revise it. With two-step rules, use the "difference method" (change in output ÷ change in input) to find the multiplier first, then solve for the constant.

🧩
Two machines in series: label the link value

When two machines are chained, write down the output of Machine A before you start Machine B. Even if the question only asks for the final answer, writing the intermediate value clearly on your working prevents you from muddling the two machines together.

🎯
BODMAS applies to algebraic forms

If a rule is written as an expression like 4x − 3, always multiply first then subtract — even if the question describes it as "multiply by 4 then subtract 3". These are the same thing, but written differently. Treat left-to-right machine diagrams as BODMAS-safe since each step is separate.
