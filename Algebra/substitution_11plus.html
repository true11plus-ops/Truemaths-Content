1

The Three Golden Rules

Before you touch a single number, these three rules must be in your head every time.

Rule 1 — Invisible Multiplication

3a means 3 × a.  xy means x × y. The multiplication sign is always hidden when a number sits next to a letter.

Rule 2 — Bracket Negatives

If you substitute a negative number, always put it in brackets. If x = −4, write (−4), not just −4. This stops sign errors.

Rule 3 — BODMAS

Brackets → Orders (powers/roots) → Division/Multiplication → Addition/Subtraction. Never skip the order.

2

Basic Substitution (Level 1)

Replace every letter with its number, then calculate carefully using BODMAS.

Worked Example
If a = 5 and b = 3, find the value of 2a + 4b.
  1. 1Substitute: 2×5 + 4×3
  2. 2Multiply first (BODMAS): 10 + 12
  3. 3Add: = 22
Second Example
If x = 6 and y = 2, find the value of 5x − 3y + 1.
  1. 1Substitute: 5×6 − 3×2 + 1
  2. 2Multiply: 30 − 6 + 1
  3. 3Left to right: = 25
⚠️
Watch Out!

Don't add before you multiply. In 2a + 4b, you cannot write (2+4) first — the terms belong to different letters.

3

Powers and Indices (Level 2)

A power only applies to the letter (or bracket) directly beneath it. Square the number before multiplying by any coefficient in front.

Worked Example
If x = 4, find the value of 3x².
  1. 1Substitute: 3 × (4)²
  2. 2Apply the power (Orders before Multiply): 4² = 16
  3. 3Multiply: 3 × 16 = 48
ApproachWorkingAnswerCorrect?
Square first ✓ 3 × (4²) = 3 × 16 48 Yes
Multiply first ✗ (3×4)² = 12² = 144 144 No!
⚠️
Watch Out!

The coefficient (the number in front) is NOT part of the power. 3x² means 3 × (x²), not (3x)².

4

Working with Negative Numbers (Level 3)

Negative numbers in substitution are where most marks are lost. Always bracket them and remember: negative × negative = positive.

Worked Example 1 — Basic negative
If p = −3 and q = 5, find the value of 2p − q.
  1. 1Substitute: 2×(−3) − 5
  2. 2Multiply: −6 − 5
  3. 3Subtract: = −11
Worked Example 2 — Squaring a negative
If y = −4, find the value of y².
  1. 1Substitute: (−4)²
  2. 2Calculate: (−4) × (−4) = 16

Negative × negative = positive. Squaring always gives a positive result.

⚠️
Watch Out!

If p = −3 and you need p², the answer is +9 not −9. Write (−3)² = 9.

5

Fractions and Division

When an expression is written as a fraction, treat the numerator (top) and denominator (bottom) as two separate mini-calculations. Find both values first, then divide.

Worked Example
If x = 10, y = 2, z = 4, find the value of (x + y) ÷ z.
  1. 1Top (numerator): 10 + 2 = 12
  2. 2Bottom (denominator): 4
  3. 3Divide: 12 ÷ 4 = 3
Second Example — Negative fractional result
If a = −5, b = 3, find the value of (2a + b) ÷ 7.
  1. 1Top: 2×(−5) + 3 = −10 + 3 = −7
  2. 2Bottom: 7
  3. 3Divide: −7 ÷ 7 = −1
⚠️
Watch Out!

Always simplify your fractional answer. If you get 8/4, write 2. If you get 3/2, write 1½ or 1.5.

6

Nested (Chained) Substitution

Sometimes one variable is defined in terms of another. Always find the inner value first, then substitute it into the main expression.

Worked Example
A = 2b + 5. If b = 3c and c = 2, find A.
  1. 1Find b first: b = 3 × 2 = 6
  2. 2Substitute b = 6 into the main formula: A = 2×6 + 5
  3. 3Calculate: 12 + 5 = 17
⚠️
Watch Out!

Always work from the inside out. Find all "helper" variables before tackling the main expression.

7

Mixed Product Terms (e.g. ab, 3a²b)

GL papers often include expressions with product terms — letters placed next to each other, like ab or 3ab. Each pair of letters means multiply those two values together. Handle these carefully alongside squared terms and BODMAS.

Worked Example 1 — Expression with ab term
If a = 3 and b = −2, find the value of 2a² − 3ab + b.
  1. 1Substitute all values: 2×(3)² − 3×(3)×(−2) + (−2)
  2. 2Deal with the power first (Orders): 3² = 9, so the expression is: 2×9 − 3×3×(−2) + (−2)
  3. 3Multiply each term: 18 − (−18) + (−2)
  4. 4Simplify signs: 18 + 18 − 2 (subtracting a negative = adding)
  5. 5Final answer: = 34
Worked Example 2 — Simple product term
If x = 4 and y = −3, find the value of xy + 2x.
  1. 1Substitute: (4)×(−3) + 2×(4)
  2. 2Multiply: −12 + 8
  3. 3Add: = −4
⚠️
Watch Out!

With 3ab, you multiply three things: the coefficient 3, the value of a, and the value of b. Write each step out separately — don't try to do it all in one go.

8

Decimal and Fractional Substitution

GL papers sometimes use decimals like 0.5 or simple fractions like ½ as the value to substitute. The same rules apply — just be careful when squaring fractions.

Worked Example 1 — Decimal value
If x = 1.5, find the value of 4x².
  1. 1Substitute: 4 × (1.5)²
  2. 2Square first: 1.5² = 1.5 × 1.5 = 2.25
  3. 3Multiply: 4 × 2.25 = 9
Worked Example 2 — Fraction value (key trap!)
If y = ⅓, find the value of 9y².
  1. 1Substitute: 9 × (⅓)²
  2. 2Square the fraction first: (⅓)² = ⅓ × ⅓ = 1⁄9  (NOT ³⁄₉ or ⅔!)
  3. 3Multiply: 9 × 1/9 = 1
Common MistakeWrong WorkingCorrect Working
Doubling the denominator (⅓)² = ²⁄₃  ✗ (⅓)² = 1⁄9  ✓
Squaring only the top (⅓)² = ¹⁄₃  ✗ Square top AND bottom: 1²/3² = 1⁄9  ✓
⚠️
Watch Out!

When you square a fraction, both the numerator and the denominator get squared. (⅓)² = 1²/3² = 1/9.

9

The −x² vs (−x)² Trap

This is one of the most common traps set deliberately on 11+ papers. −x² and (−x)² look almost identical but give different answers when x is a negative number.

ExpressionWhat it meansIf x = −3Answer
(−x)² Square the whole bracket (−(−3))² = (3)² 9
−x² Square x, then negate −(3²) = −(9) −9
Worked Example 1 — −x² (no brackets)
If x = −3, find the value of −x².
  1. 1The expression means: −(x²)
  2. 2Square first: (−3)² = 9
  3. 3Apply the negative sign: −(9) = −9
Worked Example 2 — (−x)² (with brackets)
If x = −3, find the value of (−x)².
  1. 1Work inside the bracket first: −(−3) = +3
  2. 2Square: (3)² = 9
⚠️
Watch Out!

−x² means the negative of (x squared). The power applies to x only — the minus sign waits until after. If you see −x², the answer will be negative (or zero). If you see (−x)², the answer is always positive.

10

Formulae in Context

GL Assessment questions often dress up substitution as a real-world problem using a named formula. Recognise these formulae and treat them as substitution questions — they work in exactly the same way.

Area of a triangle
A = ½ × b × h
Speed
s = d ÷ t
Simple interest
I = (P × R × T) ÷ 100
Perimeter of rectangle
P = 2(l + w)
Worked Example 1 — Area of a triangle
Find the area of a triangle with base b = 8 cm and height h = 5 cm.
  1. 1Write the formula: A = ½ × b × h
  2. 2Substitute: A = ½ × 8 × 5
  3. 3Calculate: ½ × 40 = 20 cm²
Worked Example 2 — Speed, Distance, Time
A car travels 120 km in 1.5 hours. Find its speed using s = d ÷ t.
  1. 1Identify: d = 120, t = 1.5
  2. 2Substitute: s = 120 ÷ 1.5
  3. 3Calculate: = 80 km/h
Worked Example 3 — Simple interest
Find the interest on £500 at 4% per year for 3 years. I = (P × R × T) ÷ 100.
  1. 1Identify: P = 500, R = 4, T = 3
  2. 2Substitute: I = (500 × 4 × 3) ÷ 100
  3. 3Numerator: 6000 ÷ 100
  4. 4Answer: I = £60
⚠️
Watch Out!

Real-world formula questions are still substitution questions. Don't be thrown by the story or the units — find the formula, substitute the values, and follow BODMAS.

11

Reverse Substitution — Finding the Unknown

Sometimes you are given the result of a formula and asked to find a missing variable. This is substitution run backwards — substitute the values you know, then solve for the one you don't.

Worked Example 1 — Find the unknown
Given A = 3x + 7 and A = 22, find the value of x.
  1. 1Substitute what you know: 22 = 3x + 7
  2. 2Subtract 7 from both sides: 22 − 7 = 3x → 15 = 3x
  3. 3Divide both sides by 3: x = 5
Worked Example 2 — Missing value in a formula
Using s = d ÷ t, a car travels at 60 km/h for 2.5 hours. Find the distance.
  1. 1Substitute known values: 60 = d ÷ 2.5
  2. 2Rearrange: d = 60 × 2.5
  3. 3Calculate: d = 150 km
Worked Example 3 — Two-step reverse
If 2m − 5 = 11, find m.
  1. 1Add 5 to both sides: 2m = 16
  2. 2Divide by 2: m = 8
  3. 3Check: 2×8 − 5 = 16 − 5 = 11 ✓
⚠️
Watch Out!

Always check your answer by substituting it back into the original expression. If both sides match, you're correct!

⭐ Complete Rules Summary

Every rule covered in this guide — commit these to memory before your exam.

1Invisible multiplication: 3a means 3×a, and xy means x×y.
2Bracket negatives: Always write negative substitutions in brackets to protect the sign.
3BODMAS order: Brackets → Orders (powers) → ÷ × → + −. Never swap the order.
4Powers apply to letters only: In 3x², square x first, then multiply by 3.
5Negative squared = positive: (−4)² = 16. Squaring always gives a positive result.
6Fractions — do top and bottom separately before dividing. Simplify your final answer.
7Nested substitution: Find "inner" variables first, then work outwards to the main expression.
8Mixed product terms: ab means a×b. Multiply all values in a product term one at a time.
9Fractional values: Square both numerator and denominator. (⅓)² = 1/9, not 2/3.
10−x² ≠ (−x)²: −x² means −(x²). Only brackets around −x allow you to square the negative sign.
11Formulae in context: Real-world questions are substitution in disguise. Identify the formula, substitute, calculate.
12Reverse substitution: Substitute known values, then rearrange to find the unknown. Always check by substituting back.
13Show your working: A correct substitution step can earn a method mark even if your arithmetic slips.
14Check the sign: Before writing your final answer, ask — should this be positive or negative?
