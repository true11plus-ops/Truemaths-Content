Vocabulary Foundation

Three-Way Distinction

TermDefinitionExampleWhat GL asks you to do
Expression A group of terms — no equals sign 3x + 5 "Write an expression for the perimeter." → just write the algebra, don't solve
Equation Two expressions set equal to each other 3x + 5 = 20 "Solve the equation." → find the value of the unknown
Formula A rule linking two or more variables A = l × w "Use the formula to find A." → substitute values in

Key Terms

📖
Variable A letter standing for an unknown number. In 3x + 5, the variable is x.
📖
Coefficient The number multiplying the variable. In 3x + 5, the coefficient is 3.
📖
Constant A fixed number with no variable. In 3x + 5, the constant is 5.
📖
Subject of a formula The variable on its own on one side. In A = l × w, the subject is A.

Word-to-Algebra Translation

English phraseOperationExample translation
sum of, more than, increased by, added to+"x increased by 4" → x + 4
difference, less than, decreased by, subtracted from−"3 less than x" → x − 3
product, times, multiplied by, double/triple×"double y" → 2y
quotient, divided by, shared equally, per÷"p divided by 4" → p/4
🎯
Exam Trap: Word Order in Subtraction "3 less than x" means start with x and take away 3 → x − 3. NOT 3 − x.
"10 subtracted from p" → p − 10. NOT 10 − p.
Rule: the thing being reduced comes first in the algebra.

Inverse Operations & The Balance Rule

⚖️
The Balance Rule An equation is like a set of scales — both sides are equal. Whatever operation you perform on one side, you must perform identically on the other side to keep the balance.

Inverse Operation Pairs

↔️
Addition ↔ Subtraction
Multiplication ↔ Division
Squaring ↔ Square root
Cubing ↔ Cube root
💡
Goal: Isolate the Variable Work backwards through the operations applied to x. Undo the last operation first (the one farthest from x), then the one before it.

Worked Example — Two-Step Equation

✏ Solve: 4x − 7 = 13
4x − 7 = 13 → Starting equation
4x = 20 → Add 7 to both sides (undo the subtraction of 7)
x = 5 → Divide both sides by 4 (undo the multiplication by 4)
✅
Check (always do this in the exam!) Substitute x = 5 back in: 4(5) − 7 = 20 − 7 = 13 ✓ Both sides equal — correct!

Equation Type Progression

Type 1 — One/Two-Step (no brackets, no fractions)

These are the most straightforward: apply one or two inverse operations to isolate x.

✏ Example A: Solve 3x + 8 = 23
3x + 8 = 23→Starting equation
3x = 15→Subtract 8 from both sides
x = 5→Divide both sides by 3
✏ Example B: Solve 4x − 7 = 13 (with full check)
4x − 7 = 13→Starting equation
4x = 20→Add 7 to both sides
x = 5→Divide both sides by 4

Check: 4(5) − 7 = 13 ✓

Type 2 — Equations with Brackets

Two valid methods exist. Both always reach the same answer — use whichever feels cleaner.

✏ Example A: Solve 3(2x − 1) = 15
Method 1: Expand First
3(2x − 1) = 15
6x − 3 = 15Expand: 3×2x and 3×(−1)
6x = 18Add 3 to both sides
x = 3Divide by 6
Method 2: Divide First
3(2x − 1) = 15
2x − 1 = 5Divide both sides by 3
2x = 6Add 1 to both sides
x = 3Divide by 2

Check: 3(2×3 − 1) = 3(5) = 15 ✓

✏ Example B: Solve 2(x + 5) = 18
2(x + 5) = 18→Divide both sides by 2 (cleaner here)
x + 5 = 9→Subtract 5 from both sides
x = 4→Done

Check: 2(4 + 5) = 2(9) = 18 ✓

⚠️
Common Mistake: Forgetting to Multiply All Terms When expanding, every term inside the bracket gets multiplied. 3(2x − 1) = 6x − 3, not 6x − 1. Always check both terms.
Type 3 — Unknowns on Both Sides ⭐ Most Tested

Collect all the x terms on one side and all the number terms on the other.

  • If x appears on the right, subtract the smaller x term from both sides first.
  • Then collect the number terms on the opposite side.
  • Solve the resulting one-step equation.
✏ Example A: Solve 5x + 3 = 2x + 12
5x + 3 = 2x + 12→x appears on both sides
3x + 3 = 12→Subtract 2x from both sides (removes x from right)
3x = 9→Subtract 3 from both sides
x = 3→Divide both sides by 3

Check: LHS = 5(3)+3 = 18   RHS = 2(3)+12 = 18 ✓

✏ Example B: Solve 3x + 2 = x + 10
3x + 2 = x + 10→Subtract x from both sides
2x + 2 = 10→Subtract 2 from both sides
2x = 8→Now a simple one-step equation
x = 4→Divide both sides by 2

Check: LHS = 3(4)+2 = 14   RHS = 4+10 = 14 ✓

⚠️
Common Mistake: Subtracting from the Wrong Side Always subtract the smaller x coefficient so you end up with a positive x term. In 3x + 2 = x + 10, subtract x (smaller) not 3x. If you subtract 3x you get 2 = −2x + 10 — this works but is harder and trips many students up.
Type 4 — Equations with Fractions

Always clear the fraction first by multiplying both sides by the denominator. Then solve as normal.

✏ Example A: Solve x/3 + 4 = 7
x/3 + 4 = 7→Denominator is 3 — multiply everything by 3
x + 12 = 21→3 × (x/3) = x   and   3 × 4 = 12   and   3 × 7 = 21
x = 9→Subtract 12 from both sides

Check: 9/3 + 4 = 3 + 4 = 7 ✓

✏ Example B: Solve (x + 1)/2 = 5
(x + 1)/2 = 5→Multiply both sides by 2
x + 1 = 10→Fraction cleared — the whole bracket × 2
x = 9→Subtract 1 from both sides

Check: (9+1)/2 = 10/2 = 5 ✓

💡
Always Clear Fractions First Before doing anything else, multiply both sides by the denominator. This turns the equation into a standard type you already know how to solve. Never try to work with the fraction in place — it wastes time and causes errors.

Substitution

Substitution means replacing a letter with a given number to calculate the value of an expression or formula. Always use brackets when substituting a negative number.

Level 1 — Simple Positive Single Variable

✏ C = 2d + 5. Find C when d = 4.
C = 2d + 5→Replace d with 4
C = 2(4) + 5→Multiply first (order of operations)
C = 13→8 + 5 = 13

Level 2 — Substitution with Negative Values

✏ Find the value of 3a − b when a = −2, b = 5
3a − b→Replace a with (−2) and b with 5 — always use brackets
= 3(−2) − (5)→3 × (−2) = −6 (positive × negative = negative)
= −6 − 5→Negative minus positive = more negative
= −11→Final answer
⚠️
Always Use Brackets with Negatives Writing 3(−2) makes it clear you are multiplying 3 by negative 2. Writing 3 × −2 without brackets is ambiguous and leads to sign errors. Use brackets every time.

Level 3 — Two-Variable Formula

✏ v = u + at. Find v when u = 10, a = −3, t = 4
v = u + at→Replace each variable with its value
v = 10 + (−3)(4)→a = −3, t = 4: note (−3)(4) = −12
v = 10 + (−12)→Adding a negative = subtracting
v = −2→10 − 12 = −2

Level 4 — Formula with Powers

✏ A = 3r². Find A when r = 5
A = 3r²→Replace r with 5
A = 3 × 5²→Square 5 FIRST (powers before multiplication)
A = 3 × 25→5² = 25
A = 75→3 × 25 = 75
🎯
Exam Trap: Squaring a Negative If r = −3, then r² = (−3)² = (−3) × (−3) = +9, not −9.
Negative × negative = positive. Always write the brackets: (−3)², never just −3².

Rearranging Formulae — Changing the Subject

📖
Key Concept Rearranging a formula uses exactly the same inverse-operation method as solving an equation — the only difference is that you work with letters instead of numbers. The goal is to isolate the target variable on one side.

One-Step Rearrangement

✏ Make s the subject of P = 4s
P = 4s→Want s on its own
s = P/4→Divide both sides by 4

Two-Step Rearrangement

✏ Make r the subject of C = 2r + 5
C = 2r + 5→r is buried — undo what's been done to it
C − 5 = 2r→Subtract 5 from both sides (undo +5)
r = (C − 5)/2→Divide both sides by 2 (undo ×2)

Verify: if C = 11 → r = (11−5)/2 = 3. Check: 2(3)+5 = 11 ✓

✏ Make a the subject of v = u + at
v = u + at→Want a on its own
v − u = at→Subtract u from both sides
a = (v − u)/t→Divide both sides by t
💡
Same Method as Solving Equations The process is identical — just swap the numbers for letters. Ask yourself: "What is being done to the variable I want, and in what order?" Then undo those operations in reverse order.

Forming & Solving from Word Problems

Stage 1: Define  →  Stage 2: Form  →  Stage 3: Solve  →  Stage 4: Answer in context
📋
The Four-Stage Process
  • Define: Write "Let x = …" — name your unknown clearly
  • Form: Translate the problem sentence by sentence into an equation
  • Solve: Use your equation techniques to find x
  • Answer: Write a sentence using the context (include units)

Worked Example 1 — Age Problem

✏ "Sam is 3 times as old as his brother. Their ages add up to 24. Find both ages."
  1. Define: Let the brother's age = x. Then Sam's age = 3x.
  2. Form: Together they sum to 24 → x + 3x = 24
  3. Solve:
    4x = 24→Simplify x + 3x
    x = 6→Divide by 4
  4. Answer: Brother is 6 years old. Sam is 3 × 6 = 18 years old.
  5. Check: 6 + 18 = 24 ✓

Worked Example 2 — Geometry Context

✏ "A rectangle has length (2x + 3) and width x. The perimeter is 36. Find x and the area."
  1. Define: Let width = x. Then length = 2x + 3.
  2. Form: Perimeter of rectangle = 2(length + width)
    → 2(2x + 3 + x) = 36
  3. Solve:
    2(3x + 3) = 36→Simplify 2x+3+x = 3x+3 inside brackets
    3x + 3 = 18→Divide both sides by 2
    3x = 15→Subtract 3
    x = 5→Divide by 3
  4. Answer: Width = 5, Length = 2(5)+3 = 13.
    Area = 5 × 13 = 65 square units.
  5. Check perimeter: 2(13 + 5) = 2(18) = 36 ✓

Worked Example 3 — Cost/Quantity Problem

✏ "Pens cost 30p more than pencils. Together one pen and one pencil cost £1.50. Find the cost of each."
  1. Define: Let the cost of a pencil = x pence. Then the cost of a pen = x + 30.
  2. Form: One of each costs 150p → x + (x + 30) = 150
  3. Solve:
    2x + 30 = 150→Simplify left side
    2x = 120→Subtract 30
    x = 60→Divide by 2
  4. Answer: Pencil costs 60p. Pen costs 60 + 30 = 90p.
  5. Check: 60 + 90 = 150p = £1.50 ✓
💡
Always Match Your Units In Example 3, the problem mixed pence and pounds. Convert everything to the same unit before forming the equation. £1.50 = 150p — define x in pence and work in pence throughout.

Geometry Expressions — Algebra in Shape Problems

GL papers regularly give shapes with algebraic side lengths and ask you to write expressions, form equations, and solve them.

Worked Example 1 — Rectangle

✏ Rectangle with sides (3x + 1) and (x + 2). Write the perimeter expression, then solve if perimeter = 30.
  1. Write perimeter expression:
    P = 2(3x + 1) + 2(x + 2) = 6x + 2 + 2x + 4 = 8x + 6
  2. Form the equation: 8x + 6 = 30
  3. Solve:
    8x + 6 = 30
    8x = 24Subtract 6
    x = 3Divide by 8
  4. Dimensions: Length = 3(3)+1 = 10, Width = 3+2 = 5.
  5. Check area: 10 × 5 = 50 square units. Check perimeter: 2(10+5) = 30 ✓

Worked Example 2 — Equilateral Triangle

✏ Equilateral triangle with side (2x − 4). Perimeter = 30. Find x and verify the side is positive.
  1. Form equation: All 3 sides equal → 3(2x − 4) = 30
  2. Solve:
    3(2x − 4) = 30
    2x − 4 = 10Divide both sides by 3
    2x = 14Add 4
    x = 7Divide by 2
  3. Side length: 2(7) − 4 = 14 − 4 = 10 units
  4. Positive check: 10 > 0 ✓ Valid answer.
  5. Check perimeter: 3 × 10 = 30 ✓
🎯
Exam Trap: Always Check the Side Length is Positive If x = 2 and the side is (2x − 6), the side = 2(2) − 6 = −2. A negative length is impossible — this means you've made an error. Re-check your equation. GL setters sometimes use this to catch students who solve correctly but don't verify.

Simple Inequalities — QE Top-End Awareness

🌟
Top-End Only Simple inequalities appear in the hardest questions on QE-level GL papers. The method is identical to solving equations — with one important extra rule.

Inequality Symbols

SymbolMeaningExample
>greater thanx > 3 means x is any number above 3 (not including 3)
<less thanx < 7 means x is any number below 7
≥greater than or equal tox ≥ 5 includes 5 itself
≤less than or equal tox ≤ 10 includes 10 itself

Solving an Inequality

✏ Solve: 2x + 1 < 9
2x + 1 < 9→Treat the < sign exactly like an = sign
2x < 8→Subtract 1 from both sides
x < 4→Divide both sides by 2

Answer: x can be any number less than 4 (e.g., 3, 3.5, −10, but NOT 4 itself).

⚠️
Key Rule: Multiplying/Dividing by a Negative Flips the Sign If you ever divide or multiply both sides by a negative number, the inequality sign reverses direction: < becomes >. This is awareness-level only for 11+ — avoid it by always keeping x positive where possible.

Graded Practice Questions

Work each question independently before revealing the answer. Difficulty rises from Q1 to Q8.

1
Solve: 5x + 9 = 34. Show your working and verify your answer.
Easy · Two-step
Show Answer

5x + 9 = 34

5x = 25 (subtract 9 from both sides)

x = 5 (divide both sides by 5)

Check: 5(5) + 9 = 25 + 9 = 34 ✓

2
Solve: 4(3x − 2) = 28. Show both possible methods.
Easy · Brackets
Show Answer

Method 1 (divide first):

3x − 2 = 7 (÷4 both sides) → 3x = 9 → x = 3

Method 2 (expand first):

12x − 8 = 28 → 12x = 36 → x = 3

Check: 4(3×3 − 2) = 4(7) = 28 ✓

3
Solve: 7x − 4 = 4x + 11. Collect like terms systematically.
Medium · Both sides
Show Answer

7x − 4 = 4x + 11

3x − 4 = 11 (subtract 4x from both sides)

3x = 15 (add 4 to both sides)

x = 5 (divide by 3)

Check: LHS = 7(5)−4 = 31   RHS = 4(5)+11 = 31 ✓

4
Solve: (2x − 3)/4 = 5. Clear the fraction first.
Medium · Fractions
Show Answer

(2x − 3)/4 = 5

2x − 3 = 20 (multiply both sides by 4)

2x = 23 (add 3)

x = 11.5 (divide by 2)

Check: (2×11.5 − 3)/4 = (23−3)/4 = 20/4 = 5 ✓

5
Given the formula T = 2p − q², find T when p = −3 and q = 4. Show your bracket placement clearly.
Medium · Substitution with negatives
Show Answer

T = 2p − q²

T = 2(−3) − (4)²

T = −6 − 16

T = −22

Note: (4)² = 16 (positive, because 4 is positive here). 2(−3) = −6.

6
Make u the subject of the formula v = u + at. Then make t the subject.
Medium · Rearranging
Show Answer

Make u the subject:

v = u + at → v − at = u → u = v − at

Make t the subject:

v = u + at → v − u = at → t = (v − u)/a

7
An isosceles triangle has two equal sides of length (3x + 5) and a base of (x + 2). The perimeter is 47. Find x and state all three side lengths.
Medium · Geometry word problem
Show Answer

Perimeter = 2(3x + 5) + (x + 2) = 47

6x + 10 + x + 2 = 47

7x + 12 = 47

7x = 35

x = 5

Equal sides: 3(5)+5 = 20 units each

Base: 5+2 = 7 units

Check: 20 + 20 + 7 = 47 ✓ All sides positive ✓

8
GL Multi-step: "Three friends — Amir, Bella, and Cara — collect stickers. Bella has twice as many as Amir. Cara has 8 fewer than Bella. Together they have 92 stickers."
(a) Write an expression for each person's count in terms of x (let Amir = x).
(b) Form and solve an equation to find x.
(c) State how many stickers each person has.
(d) If they share all stickers equally between the three of them, how many does each receive?
Hard · Multi-step GL-style
Show Answer

(a) Expressions:

  • Amir = x
  • Bella = 2x
  • Cara = 2x − 8

(b) Equation:

x + 2x + (2x − 8) = 92

5x − 8 = 92

5x = 100

x = 20

(c) Sticker counts:

  • Amir: 20
  • Bella: 40
  • Cara: 2(20) − 8 = 32

Check: 20 + 40 + 32 = 92 ✓

(d) Equal share: 92 ÷ 3 = 30 remainder 2 — they cannot share equally! Each gets 30 with 2 left over. (Or if the question expects a decimal: 30.67 each — state this and note it doesn't divide evenly.)
