Linear (Arithmetic) Sequences & the nth Term

📖
Definition A linear (arithmetic) sequence has a constant difference between consecutive terms — the same number is added (or subtracted) every time. This constant is called d (the common difference). The first term is called a.

Deriving the nth Term — Four Steps

The nth term formula lets you find any term in the sequence without listing all the ones before it. Here is how to build it from scratch:

  • Find d. Subtract any term from the one after it. The difference must be the same every time — check at least twice.
  • Find the "0th term." The 0th term is the imaginary term before the first one. Calculate it as: 0th term = a − d.
  • Write the formula. nth term = d × n + (0th term).
  • Verify. Substitute n = 1, 2, 3 back in and check the formula gives the correct sequence terms.

Reverse nth Term — "Which Term Has Value X?"

GL papers regularly give you the nth term formula and ask: "Which term in the sequence equals 47?" The method is to set the formula equal to the target value and solve for n — exactly like solving an equation.

Set: nth term formula = target value
Solve for n (must be a positive whole number)

Worked Example 1 — Increasing Sequence

✏ "Which term in the sequence 3n + 2 has value 47?"
  1. Set formula equal to target: 3n + 2 = 47
  2. Subtract 2: 3n = 45
  3. Divide by 3: n = 15

The 15th term equals 47.

Check: 3(15) + 2 = 45 + 2 = 47 ✓

Worked Example 2 — Decreasing Sequence

✏ "Which term in 23 − 3n has value 8?"
  1. Set: 23 − 3n = 8
  2. Rearrange: 23 − 8 = 3n → 15 = 3n
  3. Divide: n = 5

The 5th term equals 8.

Check: 20, 17, 14, 11, 8 ✓ (5th term)

🎯
Critical Check: n Must Be a Positive Whole Number If your answer for n is a decimal (e.g., n = 7.3) or negative, that value is not in the sequence. Either the question is testing whether you recognise this, or you've made an arithmetic error. Re-check your working before writing a final answer.
💡
GL Extension: "Does 100 appear in the sequence 4n − 1?" Set 4n − 1 = 100 → 4n = 101 → n = 25.25. Since 25.25 is not a whole number, 100 does not appear in this sequence.

Non-Linear nth Term Formulas

Not every sequence increases by the same amount each time. When the differences are not constant, look at the second differences (differences of differences) to identify the type.

The Second-Difference Diagnostic

🔬
How to Read Second Differences
  • First differences constant → Linear sequence (dn + c)
  • Second differences constant = 2 → Square-based (contains n²)
  • Second differences constant = 6 → Cube-based (contains n³)

Finding the coefficient of n² When 2nd differences = k (constant), the coefficient of n² is always k ÷ 2. So 2nd diff = 4 → coefficient = 2. 2nd diff = 2 → coefficient = 1 (pure n²).

✏ "Find the 8th term of 3, 6, 12, 24… (r = 2)"

💡
The ar^(n−1) formula The general formula is T(n) = a × r^(n−1), where a = first term. For the example above: T(8) = 3 × 2⁷ = 3 × 128 = 384. This is beyond the 11+ syllabus, but the table method above gives the same answer and is always acceptable.

✏ Worked Example: Machine is ×3 then −1. Output = 17. Find the input.
  1. Reverse the last step first: 17 + 1 = 18
  2. Reverse the first step: 18 ÷ 3 = 6

Check forward: 6 × 3 = 18, 18 − 1 = 17 ✓

⚠️
Common Mistake: Reversing in the Wrong Order Always reverse the last operation first, working backwards through the machine. Students who reverse operations in forward order get the wrong answer.

Direction 3 — Find the Rule from a Table

✏ Derive the rule from this table:
Input (x)1234
Output (y)57911
  1. Output differences: 7−5=2, 9−7=2 → constant = 2 → rule contains 2x.
  2. Test 2x: when x=1, 2×1=2. But output is 5. Gap = 5−2 = +3.
  3. Rule: y = 2x + 3
  4. Verify: x=4 → 2(4)+3 = 11 ✓

Direction 4 — Unknown Operation

✏ "The machine turns 4 into 11 and 7 into 17. What is the rule?"
  1. Find the output difference: 17 − 11 = 6. Find the input difference: 7 − 4 = 3.
  2. Output grows twice as fast → rule contains 2x.
  3. Test 2x at x=4: 2(4) = 8. But output is 11. Gap = 11 − 8 = +3.
  4. Rule: y = 2x + 3
  5. Verify: x=7 → 2(7)+3 = 17 ✓
🎯
Exam Trap: The Multiplier First Always find the multiplier (coefficient of x) before the constant. Output difference ÷ input difference = multiplier. Then test to find what to add or subtract.

Diagnostic Strategy — Identify Any Sequence

When you encounter an unknown sequence in the exam, run through this decision tree in order:

Step 1 — Find 1st differences
Subtract consecutive terms. Are they constant?
YES → Linear sequence. Find nth term = dn + c.
Step 2 — Find 2nd differences
If 1st differences aren't constant, find differences of differences.
= 2 → Square-based (n²). = 6 → Cube-based (n³).
Step 3 — Check ratio
Is each term the same multiple of the previous one?
YES → Geometric. Find r = term₂ ÷ term₁.
Step 4 — Check previous two
Does each term equal the sum of the two before it?
YES → Fibonacci-type. Extend term by term.
Step 5 — Odd/even split
No single rule working? Split into positions 1,3,5… and 2,4,6…
→ Alternating/interwoven sequence.
Step 6 — Two-operation
Try: is there a ×then+ or ×then− pattern?
Test on first three terms. If consistent: two-operation rule found.
🗺️
Summary in One Line per Type
  • Constant 1st diff → Linear (dn + c)
  • 2nd diff = 2 → n²-based; 2nd diff = 4 → 2n²-based
  • Constant ratio → Geometric
  • Each = sum of previous two → Fibonacci
  • Two interleaved rules → Split odd/even positions
  • 1st diff increases by 1 → Triangular n(n+1)/2

Graded Practice Questions

Work independently before checking answers. Difficulty rises from Q1 to Q8.

1
Find the nth term of the sequence: 7, 12, 17, 22, 27…
Easy · Linear nth term
Show Answer
  1. d = 12 − 7 = 5
  2. 0th term = 7 − 5 = 2
  3. nth term = 5n + 2
  4. Verify: n=1 → 7 ✓, n=2 → 12 ✓, n=3 → 17 ✓
2
Find the nth term of the sequence: 30, 25, 20, 15, 10…
Easy · Decreasing nth term
Show Answer
  1. d = 25 − 30 = −5 (negative — sequence decreasing)
  2. 0th term = 30 − (−5) = 30 + 5 = 35
  3. nth term = −5n + 35
  4. Verify: n=1 → 30 ✓, n=2 → 25 ✓, n=6 → −30+35 = 5 ✓
3
The nth term of a sequence is 4n − 1. Which term of the sequence has the value 63? Does the value 50 appear in this sequence?
Easy · Reverse nth term
Show Answer

Part 1: 4n − 1 = 63 → 4n = 64 → n = 16. The 16th term = 63.

Part 2: 4n − 1 = 50 → 4n = 51 → n = 12.75.

12.75 is not a whole number, so 50 does not appear in this sequence.

4
For the sequence 2, 8, 18, 32, 50…: (a) What type of sequence is it? (b) Write the nth term formula. (c) Find the 12th term.
Medium · Non-linear
Show Answer

1st differences: 6, 10, 14, 18 (increasing — not linear)

2nd differences: 4, 4, 4 → constant = 4 → square-based

Coefficient of n² = 4 ÷ 2 = 2. Try 2n²: 2, 8, 18, 32, 50. This matches exactly!

(a) Modified square sequence

(b) nth term = 2n²

(c) T(12) = 2 × 144 = 288

5
Find the 7th term of the geometric sequence: 5, 15, 45, 135…
Medium · Geometric
Show Answer

r = 15 ÷ 5 = 3

Build the table:

5 → 15 → 45 → 135 → 405 → 1215 → 3645

The 7th term = 3645.

(Using formula: 5 × 3⁶ = 5 × 729 = 3645 ✓)

6
Using the formula for triangular numbers, find T(15). Why is it impossible to answer this question by listing?
Medium · Triangular formula
Show Answer

T(n) = n(n+1) ÷ 2

T(15) = 15 × 16 ÷ 2 = 240 ÷ 2 = 120

Listing would require writing all 15 triangular numbers one by one — slow and error-prone under exam conditions. The formula gives the answer in two steps.

7
A two-step function machine applies ×4 then +2. An output of 26 is produced. What was the input? Show the reverse working clearly.
Medium · Reverse function machine
Show Answer

Reverse the last operation first:

  1. Output = 26. Reverse "+2": 26 − 2 = 24
  2. Reverse "×4": 24 ÷ 4 = 6

Input = 6.

Check forward: 6 × 4 = 24, 24 + 2 = 26 ✓

8
Consider the sequence: 3, 7, 13, 21, 31…
(a) Identify the sequence type using differences.
(b) Write the nth term formula.
(c) Which is the first term of this sequence to exceed 150?
Hard · Multi-step GL-style
Show Answer

(a) Identify the type:

1st differences: 4, 6, 8, 10 — increasing, not constant.

2nd differences: 2, 2, 2 → constant = 2 → square-based sequence.

(b) Find the nth term:

Coefficient of n² = 2 ÷ 2 = 1. Try n²: 1, 4, 9, 16, 25. Compare: our sequence is 3, 7, 13, 21, 31.

Differences from n²: 3−1=2, 7−4=3, 13−9=4, 21−16=5, 31−25=6. These differences are n+1.

So nth term = n² + (n+1) = n² + n + 1.

Verify: n=1 → 1+1+1=3 ✓, n=3 → 9+3+1=13 ✓, n=5 → 25+5+1=31 ✓

nth term = n² + n + 1

(c) First term exceeding 150:

Set n² + n + 1 > 150 → n² + n > 149.

Test values: n=12 → 144+12+1=157 > 150. n=11 → 121+11+1=133 ≤ 150.

The 12th term (value 157) is the first term to exceed 150.
